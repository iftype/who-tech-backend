## CI 빌드 방식으로 배포 전환

**문제**: Oracle 무료 티어 1GB RAM에서 `npm run build` (tsc) 실행 시 OOM → PM2 강제 종료

**변경**:

- `deploy.yml`: GitHub Actions에서 빌드 → rsync로 `dist/` 서버 전송 → 서버에서 reload만
- `app.ts` `/admin/deploy`: `npm run build` 제거 — git pull + migrate + pm2 reload만 실행

**필요한 GitHub Secrets** (Actions → production environment):

- `SSH_PRIVATE_KEY`: base64로 인코딩된 서버 SSH 개인키 (`base64 -i ~/.ssh/ssh-key-oracle.key`)
- `SERVER_HOST`: `168.107.51.150`
- `SERVER_USER`: 서버 접속 유저명
- `ADMIN_SECRET`: 기존 등록됨
